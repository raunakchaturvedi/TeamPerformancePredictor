# 🚀 Team Performance Predictor (TeamLytics)

A web-based system to analyze employee burnout, measure employee performance, and predict overall team performance in an IT company.

---

## 📌 Project Overview

The **Team Performance Predictor** helps organizations:

- Identify employee burnout risk
- Evaluate employee performance
- Predict team performance
- Support management decision-making

---

## 🎯 Objective

The main objective of this project is to evaluate employees based on burnout risk and performance metrics, and then predict the overall performance of a team.

---

## 🧠 Key Concepts

### 🔴 Burnout Risk

Burnout risk shows how stressed or overloaded an employee is.

#### Factors:
- Working Hours  
- Overtime Hours  
- Stress Level  
- Leave Count  

#### Formula:

#### Classification:
- High  
- Medium  
- Low  

---

### 🟢 Employee Performance

Employee performance shows how well an employee is performing.

#### Factors:
- Tasks Completed  
- Deadlines Met  
- Productivity Score  
- Teamwork Score  

#### Formula:

#### Classification:
- High  
- Medium  
- Low  

---

### 🔵 Team Performance Prediction

Team performance is calculated using both burnout and performance data.

#### Formula:

#### Result:
- Score ≥ 2 → High  
- Score ≥ 0 → Medium  
- Score < 0 → Low
